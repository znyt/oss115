　　第373章
<br/>　　这是在黎明开始的一场战斗。
<br/>　　守方是帝国方面的青州兵，在与正义教的交锋中，帝国军一败涂地，太多人被俘，太多人被杀，逃的人溃不成军，追的人如狼似虎。
<br/>　　青州方面的指挥官名叫韩若离，他是新近被提拔上来的帝国少将，等待他的是一眼望不到头的官运亨通。可这场溃败却直接将他的官运扼杀在摇篮中，韩若离没有想到，这些面朝黄土的刁民，在拿起武器后，竟爆发出难以想象的战斗力。将青州和并州两方面十五万联军，一举击溃。
<br/>　　韩若离在溃逃的时候组织败兵，终于聚拢起一支五千人的包围，然而，却陷入了正义教的包围中。
<br/>　　几次突围都被敌军挡了回来，同样，正义教组织的几次进攻，也没有取到什么效果。双方的僵持已经持续三天。
<br/>　　在这三天内，双方已不知交锋多少次，死伤无数。兵困马乏，如今也只是在苦苦支撑而已，而韩若离也明白，己方其实已支撑不了多久。
<br/>　　无非一死而已。
<br/>　　韩若离握紧手中的达大刀，发出一声狂吼。
<br/>　　“冲啦！”
<br/>　　一队败兵鼓起最后的战意，向敌军发出冲锋。正在此时，远处杀出一捎人马，为首之人黑衣黑甲，身后一杆大旗，逆风猎猎而舞。
<br/>　　杀入乱兵丛中，如同一条黑龙相仿，奔着韩若离的帅旗而来。
<br/>　　韩若离认出了对方，甚至在很久之前，他就认识林少羽。那时候，林少羽还不过是个流着鼻涕的跟屁虫，当时谁也不会想到，他竟然第一个向帝国举起反旗。
<br/>　　同时，韩若离也向对方发起冲锋，二人速度都极快，成了战场上移动的两条黑线。
<br/>　　瞬间，二人就碰在了一起，几十秒的冲锋就是为了这一刹那的相遇。然后只是一招，便分出了胜负，长枪如蛇，洞穿韩若离的胸膛。
<br/>　　韩若离瞳孔扩大，紧跟着眼中神采消失，一头跌落马下。
<br/>　　林少羽浑身浴血，将长枪举起，队伍在他面前汇聚。
<br/>　　“杀！”
<br/>　　在挑落韩若离后，战斗的胜负已经注定，在战神旗下，杀气冲斗牛，青州兵丢盔弃甲，或被杀或被俘。
<br/>　　夕阳西下，持续一天的战斗进入尾声，林少羽横枪立马，落日在他身上打下一场金黄。由这一战，正义教在幽州的地位已无法撼动。
<br/>　　……
<br/>　　初春三月，万物复苏，草原之上，野草从黑泥中泛出新绿。
<br/>　　程大雷骑在黑牛身上，深吸一口气，顿觉神清气爽。
<br/>　　樊梨花瞥了他一眼，道：“你为什么要带我来这里？”
<br/>　　“出来吹吹风么，在屋里待着会关出毛病来的。”
<br/>　　时间已过去太久，但樊梨花仍旧住在琴川关，只因为她的抑郁症还没有好转。整天黑着一张脸，长吁短叹的，本来天真烂漫，现在快成了葬花的林黛玉。
<br/>　　所以今天程大雷带着她出了琴川关，来到草原上，放松一下心情。
<br/>　　“你好像很开心？”樊梨花问。
<br/>　　“这个……很明显么，哈哈。”
<br/>　　樊梨花看程大雷一张丑脸上挤出的笑纹，轻轻摇了摇头。
<br/>　　“唉，有时候我还没羡慕你的，没心没肺，就能活得开心点。”
<br/>　　“……”程大雷。
<br/>　　怎么听樊梨花话里的意思，感觉自己像个傻瓜。与樊梨花正好相反，程大雷最近心情的确不错。幽州的情报已传到这里，只听说帝国军大败，正义教已经稳住了幽州的局势。
<br/>　　程大雷自然为林少羽开心，言谈举止中，难免就把心情流露在外面。本来是打算用这心情感染一下樊梨花，可看樊梨花的意思，似乎没什么用。
<br/>　　程大雷本来就不适合开导别人，蛤蟆寨最会做思想工作的是刘悲。可程大雷实在不敢将刘悲放在樊梨花面前，那不是以毒攻毒，反而是自找苦吃。
<br/>　　于是这件事只能由程大雷亲力亲为，每天陪着樊梨花聊聊天，散散心，樊梨花看见自己时倒也能说上两句话，若是平常怕是饭也不会吃。
<br/>　　如果是之前，帝国很少有人敢涉足草原的，这里的灰狼、黄豹可以轻易扼杀一个人的生命，更不用提戎族和帝国的深仇大恨，以及层出不穷的马匪。
<br/>　　但自从琴川关开张之后，两族商业往来不断，互相畏惧之心已渐渐消了。程大雷曾经组织人手，对草原上的马匪进行过一次大扫荡，他们有的被直接诛杀，有的逃到别的地方去。如今这片草原还算得上太平。
<br/>　　而今天，程大雷和樊梨花就是来打猎的。
<br/>　　忽地，一直野鹿从眼前一闪而逝，程大雷眼前一亮，握住了匹夫剑。以速度相比，程大雷自然不能和野兽相比，而在弓箭上，程大雷也没有什么天赋。不过，他身上带着匹夫剑、连击驽，座下黑牛速度也是奇快，想要抓几只猎物，也并不困难。
<br/>　　正在程大雷准备追击的时候，樊梨花拦住了他，程大雷一怔，不解其意。
<br/>　　“放过它吧，看着蛮可怜的。”
<br/>　　程大雷无语，这是樊梨花应该说出来的话么，杀人她都不在意，何况是杀一个人。
<br/>　　“听说鹿肉……很补的。”
<br/>　　“你很需要补么？”樊梨花冷哼一声，紧跟着又叹口气道：“现在是野鹿发情的季节，它们也在寻找配偶，你何必为了自己，杀了它们。”
<br/>　　呃……这话怎么听，怎么感觉不对。
<br/>　　正想着，一支羽箭凭空而来，射入那只野鹿的眼中，它慌不择路，竟向程大雷奔来，半程又中了一箭，一命呜呼。
<br/>　　樊梨花皱起眉头，脸上露出不忍之色。程大雷倒是不觉得如何，他的注意力全部在射箭之人身上。
<br/>　　是一伙戎族，大概有十几个人，快马奔到程大雷面前时停住。
<br/>　　“你们是那个部落的？”程大雷问。
<br/>　　“原来是程当家，我们是黑犬部穆野大人的手下。”
<br/>　　草原上，有大大小小数百个部落，大的有上万人，小的可能只有几百人。他们逐水而居，也没个首领。
<br/>　　“这头鹿程当家如果喜欢，我们就送给程当家了。”
<br/>　　程大雷如今在草原上也有些名气。他目光看过去，人群中有一个人引起了他的注意。
<br/>　　金发，蓝眸，肤白如纸。
<br/>
