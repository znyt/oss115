　　但这，并不是她应该操心的事情，只要将消息禀报上去，组织高层自然会派人前来处理的。
<br/>　　“当然可以，但需要先生随我检验完邀请函之后，我才能给您提供服务。”胡桃夹子浅笑一声，语气有些小心，似乎是生怕得罪了两人一般。
<br/>　　嘀咕了一声，刘哔也没有反对，顺从的跟在胡桃夹子后面，向左前方走去。
<br/>　　片刻，几人来到一个单独隔离出来的小房间中，屋内坐着一个头发花白的老人。
<br/>　　听到有人走了进来，老人微微抬起了眼皮，看了一眼后，漫不经心的说道：“邀请函，拿来核对一下。”
<br/>　　虽然进入的时候已经将邀请函检查了一遍，但包厢的客人依旧需要再次接受检查。
<br/>　　这是拍卖场从创建之初就遗留下来的规矩，毕竟能进入包厢的客人，每一位都是颇有身份的喰种，如果出了什么意外，即便是小丑组织也很难交代。
<br/>　　尤其是这一次，今天的拍卖会不同以往，有很多极其珍贵的东西会展示出来，能够获得这次拍卖会邀请函的人，要么就是财力雄厚，或者说身份地位极高的喰种。
<br/>　　而对于会场的安全，他们自然也比往常严厉了许多。
<br/>　　刘哔的邀请函，是从月山习手中弄到的，肯定没有什么问题，老人拿过邀请函仔细核对了一下，递还给了刘哔。
<br/>　　“这两位客人就由你来招待，记得解释一下会场的规矩。”
<br/>　　既然邀请函没有什么问题，老人自然也不会多说什么，交代了胡桃夹子一声，眯上了双眼。
<br/>　　胡桃夹子对老人恭敬的弯了弯腰，随即向刘哔两人做了一个请的手势。
<br/>　　几人向邀请函上面标注的包厢走去，路上，刘哔一边走，一边和胡桃夹子搭着话。
<br/>　　“女士，请问现在能向你打听消息了吗？”
<br/>　　语气虽然轻松，但刘哔心中却有些心急，他知道拍卖会是不会安全进行下去的，如果不提前寻找的话，以后也不知道还有没有机会找到。
<br/>　　如果没有从教皇那里得到消息，他倒是还不会感觉这么紧迫，毕竟他改变了太多剧情，CCG不一定会插手这件事情。
<br/>　　但，实际情况肯定不会像他很早之前设想的那样简单。
<br/>　　CCG是和修家族控制的，小丑组织的原首领是和修家的子弟，那么，和修家族又怎么会不知道这处地方？
<br/>　　拍卖会被搅，这绝对不是一个偶然，也不存在泄不泄露消息，从一开始和修家族就准备对小丑组织动手了。
<br/>　　“当然可以，请问先生需要询问什么事情，小女子如果知道，肯定不会隐瞒。”
<br/>　　听到刘哔再次要求，胡桃夹子便也不再推脱了，事实上对于这里，也没有几个人能比她还要熟悉。
<br/>　　“BigMadam，原MADAM喰种餐厅的主席，和月山家族似乎有些关系，你有听说过这个人吗？”
<br/>　　三人顺着长廊慢慢走着，脚步并不着急，但最终还是走到了包厢门口，等待开门的时候，刘哔问出了自己的问题。
<br/>　　胡桃夹子开门的动作稍稍顿了一下，这个动作，刘哔捕捉到了。
<br/>　　“BigMadam啊，我知道他，身为男性喰种，却总喜欢装成人类贵妇的样子，也挺让人讨厌的呢。”胡桃夹子拧动包厢的把手，露出其中极尽奢华的场景：“你们是要找他吗，我不能告诉你他在八号包厢。”
<br/>　　屋内水晶般的灯光，洋洋洒在胡桃夹子的身上，像是给她裸露在外的肌肤镀上了一层闪闪发亮的外衣，刘哔楞了一下，她的回答有些出乎自己的意料。
<br/>　　是想要将自己置身事外吗，那没问题，本来就不关她的事。
<br/>　　舒展开眉头，刘哔嘴角挂起一抹笑容：“真可惜，你不告诉我的话，那我该如何知道他在八号包厢呢？”
<br/>　　“实在不好意思，两位就只能自己碰运气寻找了。”胡桃夹子站在门口，做了一个‘请’的手势，继续说道：“看样子，两位是第一次来我们拍卖行，这样的话，那请允许我来讲解一下会场的规则。”
<br/>　　“这次拍卖会，基本上是我行成立以来，规模最大的一次喰种拍卖会，为了这次活动，我们提前准备了很久，也邀请来了很多身份高贵的客人。”
<br/>　　“理论上来说，我们对所有客人都是一视同仁的，只要你是我们邀请来的客人，就会得到我们的庇护，拍卖会举行期间，我们会负责保护两位的人身安全。”
<br/>　　给两人倒上了咖啡，胡桃夹子立在一边，将话题从BigMadam身上偏离了出去。
<br/>　　如同刘哔心中所想，她可不愿意掺和进去这些事情。
<br/>　　“但理论也只是理论，我觉得有必要提醒一下两位，如果有喜欢的商品，竞拍也请量力而为，否则如果出了什么问题，我们不会负任何责任。”
<br/>　　“就如同先生在楼下击杀的那些喰种，不是到现在都还好好的坐在这里吗？”
<br/>　　门外，传来一道阴柔的声音，惹的几人转头，看到了正在进门的一个青年男子。
<br/>　　这名男子戴着一张奇异的全覆盖式面具，身材修长，穿着一套紧身的黑色晚礼服，没露出一点皮肤。
<br/>　　“诗先生。”胡桃夹子恭敬的弯下了腰，目光深处掠过一抹惊惧。
<br/>　　这个男人的实力，她心中自然清楚，在组织中，基本上没有喰种敢违背去这个男人的意思。
<br/>　　“好了，你退下吧，这位客人，暂时由我来接待。”挥了挥手，诗坐在了两人对面，自顾拿起了一个杯子，给自己添上了一杯咖啡。
<br/>　　对于诗的吩咐，胡桃夹子自然求之不得，欠了一下身后，迅速退了出去。
<br/>　　“砰。”
<br/>　　包厢的大门关闭，房间里就只剩下了三人，刘哔斜靠在沙发上，饶有兴致的看着坐在自己对面的男人。
<br/>　　他在猜测，这个家伙的身份，但是从这身打扮和称呼上面，他大致也知道了是什么人。
<br/>　　穿的这么严实，甚至连脸都不敢露出一点，大概是想要遮盖住自己满身的刺青吧。
<br/>　　曾经的四区领导者‘呗’，现任小丑内部成员‘诗’。
