　　“难道...难道....”
<br/>　　白家主满脸希冀之色，目光灼灼地盯着萧逸的炼药炉。
<br/>　　萧逸淡淡一笑，轻轻一挥手，炉内的紫炎瞬间消散。
<br/>　　而后便是一股强烈的冰冷气息笼罩了整个场地，修为较弱者，甚至直接打了个冷颤。
<br/>　　“如此强烈的冰属性气息，难道...雪魄丹炼制成功了？”叶铭、方鹤、慕容胤三人也是直直地盯着萧逸的炼药炉。
<br/>　　“呵。”萧逸轻笑一声，看向白家主，道，“幸不辱命，雪魄丹已成。”
<br/>　　“真的成功了？”白家主脸上止不住的喜意。
<br/>　　只见萧逸再次一挥手，两粒通体雪白，浑圆剔透的丹药从炼药炉中飞出。
<br/>　　“果真是雪魄丹。”叶铭三人先是一惊，而后惊呼道，“怎么有两粒？”
<br/>　　“两粒雪魄丹？”白家主也吃了一惊。
<br/>　　萧逸耸耸肩，对白家主说道，“也不知白家主女儿的反噬情况如何，若是反噬太过严重，一颗雪魄丹无法完成抵消掉，那便麻烦了。”
<br/>　　“所以易某索性便炼制两粒，以保白家主女儿无虞。”
<br/>　　“当然，用了两份材料。”
<br/>　　萧逸笑笑，挪揄道，“怎么？看白家主如此惊讶的神情，莫不是也像旁人那般，以为易某刚才是在斗气，浪费材料不成？”
<br/>　　白家主脸上的尴尬之色一闪而过，讪笑道，“当然没有，我自然是相信易霄小兄弟的。易霄小兄弟乃人中龙凤，天赋过人，心智成熟，岂会做那等斗气之事。”
<br/>　　“原来如此。”萧逸笑笑，道，“想来也是，白家主何等人物，岂会误会我呢。”
<br/>　　“那是。”白家主大笑一声。
<br/>　　萧逸将两粒雪魄丹交于白家主，道，“白家主还是先拿丹药去救令媛吧。”
<br/>　　“若是两粒不够，以白家主的能耐，既然能在短时间内凑齐八份材料，怕是再去多找几份也是不难的。”
<br/>　　“易某再次为白家主开炉炼药便是。”
<br/>　　白家主客气地拱拱手，道，“如此便谢过易霄小兄弟了。”
<br/>　　说罢，白家主拿过两粒雪魄丹，瞬间消失在原地。
<br/>　　白家主离去，但在场武者的脸色，却全都变了。
<br/>　　看向萧逸的目光，更是充满了惊骇。
<br/>　　“天啊，易霄竟然真的炼制成功了。”
<br/>　　“雪魄丹，就连五品炼药师都不敢说一次就成功。他倒好，直接就炼制了两粒。”
<br/>　　“这岂不是说，易霄的炼药本事，比叶铭他们三人还要强？”
<br/>　　“你们看到没有，白家主都对他客客气气的。”
<br/>　　“可不是嘛，白冰雪乃是白家主的掌上明珠，这次得救，白家主自然会好好报答他。”
<br/>　　在场武者议论纷纷，在他们看来，如今的萧逸，不仅是打败了无忧公子的武道天才；更是年轻一辈中炼药术首屈一指的炼药师。
<br/>　　武道天才，天才炼药师，无论哪一个头衔，都足以惊人。
<br/>　　当两个头衔同时聚集在一人身上时，这已经不能用天才去形容了，简直是个妖孽。
<br/>　　一旁的叶铭三人听着在场武者的对话，叶铭和方鹤无奈地摇摇头，但却只是轻笑了几声。
<br/>　　只有慕容胤，满脸不忿之色。
<br/>　　这时，慕容杀走到他身边，低声问道，“慕容胤执事，炼制成雪魄丹的是易霄。我们之前拜托白家主之事，应该是难成了。”
<br/>　　“该死。”慕容胤冷冷地盯着萧逸，眼中尽是杀意。
<br/>　　“可恶的小子，坏了我的大事。”
<br/>　　事实上，萧逸并没有坏他的事，只不过他技不如人罢了。
<br/>　　就算萧逸这次没有成功，他也同样只会失败。
<br/>　　但萧逸的成功，却让得他心生妒忌，更将所有责任推到萧逸身上。
<br/>　　“那个易霄，我怎么感觉好像有些眼熟....”
<br/>　　慕容轩和慕容千军忽然疑惑地看着不远处的萧逸，喃喃自语着。
<br/>　　“哦？”慕容杀问道，“怎么，你们认识那易霄？”
<br/>　　两人同时点点头，道，“他戴着面具，无法确定，但看起来，有些像我们以前认识的一个小废物。”
<br/>　　“小废物？”慕容杀和慕容胤都愣了愣。
<br/>　　正当他们想说些什么时，忽然，一阵冰冷的气息笼罩了他们。
<br/>　　白管事凭空出现在他们身旁，冷声问道，“几位的意思是，救我白家千金的易霄小兄弟，是个废物？”
<br/>　　“阿？”慕容杀四人顿时一惊，连忙道，“白管事误会了，我们只是随便聊聊罢了。”
<br/>　　“哼。”白管事冷哼了一声。
<br/>　　一旁，显然也有些武者听到慕容杀四人刚才的对话，顿时面露嘲讽笑意。
<br/>　　“慕容胤执事，易霄此等超级天才，在你们口中，竟然是个小废物？”
<br/>　　“呵呵，真不知你们慕容家判断别人天赋的标准竟然如此高。”
<br/>　　“只不过，若易霄是废物的话？那么慕容胤执事连雪魄丹都炼制不出来，又算什么？”
<br/>　　这次来白家的武者，有不少是洞玄境中有名的武者，自然不用怕慕容胤等人，也不卖他们面子。
<br/>　　武者，追求武道，大多是率性而为之辈，有什么说什么。
<br/>　　“你们....”慕容胤立刻脸色发黑，却又无法发作。
<br/>　　这时，慕容轩和慕容千军还想说些什么，慕容杀连忙呵斥道，“闭嘴，还嫌不够丢人吗？”
<br/>　　半晌，白家主回来，脸上尽是轻松和笑意，看来两粒雪魄丹是起效了。
<br/>　　白家主径直走到萧逸身边，而后认真地拱了拱手，道，“易霄小兄弟，小女的反噬已被抵消，已然无碍。”
<br/>　　“恭喜白家主。”萧逸淡淡一笑。
<br/>　　“恭喜白家主，贺喜白家主。”
<br/>　　在场武者纷纷行了一礼。
<br/>　　“诸位。”白家主高声道，“今日小女康复，除了易霄小兄弟炼药之术过人外，也仰仗诸位的鼎力支持。”
<br/>　　“今日，还请诸位给白某一个面子，留在白家吃顿宴席。”
<br/>　　白家主是四季城的城主，本身又是白家的家主，极善和各方势力打交道。
<br/>　　所以人缘极好，这次也众多武者前来。
<br/>　　“叨扰白家主了。”众武者纷纷一笑。
<br/>　　这时，气氛极好。
<br/>　　“易霄小兄弟，今日便陪我多喝几杯。”白家主亲和地对萧逸说道。
<br/>　　众武者顿时面露羡慕之色，能和白家主同席喝酒，可是一件非常荣幸之事。
<br/>　　萧逸却摇摇头，道，“谢过白家主了，易某还有要事，恐怕吃不上这顿宴席了。”
<br/>　　“哦？何事如此急，可需我帮忙？”白家主问道。
<br/>　　“小事罢了，不劳白家主费心，易某自己能解决。”萧逸淡淡地说道。
<br/>　　白家主见萧逸确实不想多留，便没有强留，道，“易霄小兄弟，这次练成雪魄丹，救小女一命。按照之前我答应的，许你一个承诺。”
<br/>　　“你放心，只要是我白家能做到的，绝不推辞。”
<br/>　　白家主认真地保证道。
<br/>　　萧逸思索了一下，道，“不知白家主府上，可有六宝玲珑参？”
<br/>　　“六宝玲珑参？”白家主皱了皱眉头。
<br/>　　一些八卦的武者听到，也是一惊，“六宝玲珑参？”
<br/>　　“那可是四品巅峰药材阿，能凭空提升武者修为境界，连洞玄武者都趋之若鹜，梦寐以求。”
<br/>　　“可惜，那是最稀罕的天材地宝之一，四季城已经多年没有出现过了。”
<br/>　　八卦的武者们听到六宝玲珑参六字，明显双目放光。
<br/>　　白家主沉声道，“易霄小兄弟，想必你也知道这东西的稀罕程度。并非我小气，实在是我白家也没有。”
<br/>　　萧逸微微失望，但其实也早就料到了。
<br/>　　六宝玲珑参在四品天材地宝中，就如同妖血炎心果在三品天材地宝中一样，非常珍稀，可遇而不可求。
<br/>　　就连白家都没有。
<br/>　　白家主继续道，“这样如何，我发动白家的势力，尽力为你找寻，若能遇到，马上通知你。”
<br/>　　“好，有劳白家主。”萧逸点点头。
<br/>　　“只是，我寻到后，该如何找你？”白家主问道。
<br/>　　“简单。”萧逸笑道，“只需到猎妖殿花个数百两，发一道寻我的消息，易某会立刻赶来白家。”
<br/>　　“好。”白家主点头道，“明日我便派人打听寻找，一有消息，立刻通知易霄小兄弟。”
<br/>　　“谢了。”萧逸道谢一声，而后离开。
<br/>　　然而，他刚走没几步，便想起了什么，转过身，道，“白家主，可否帮易某一个小忙？”
<br/>　　“尽管说。”白家主道。
<br/>　　萧逸说道，“易某有事在身，就不去百武城交任务了。能否让白家主派人去结了任务。”
<br/>　　只要发布任务者宣布完成任务，猎妖殿的人会自动为完成任务的猎妖师记录任务点。
<br/>　　“呵呵，小事一桩罢了。”白家主笑道。
<br/>　　“如此，易某告辞。”萧逸说罢，一个转身，快速离开了白家。
<br/>　　这时，另一边，一直黑着脸的慕容胤见萧逸已经离开，这才腆着脸走到白家主身旁。
<br/>　　“白家主。”慕容胤行了一礼，道，“不知之前答应在下之事....”
<br/>　　白家主摆摆手，笑道，“慕容胤执事说的是百武城盛宝商行在四季城开分行之事吧。”
<br/>　　白家主继续道，“虽然慕容胤执事没能成功炼制雪魄丹，但我也承你的请，此事，便...”
<br/>　　白家主刚想说可以，一旁的白管事却走了上来，在他耳边耳语了几句。
<br/>　　“哦？竟有此事？”白家主顿时眉头一皱，道，“慕容胤执事，原来在你眼中，易霄小兄弟这等超级天才，竟是如此不堪，废物一个。”
<br/>　　“呵呵，看来我白家在你眼中也不过如此。你在四季城开设分行之事，就此作罢。”
<br/>　　白家主愤愤地挥了挥衣袖，再也没管慕容胤，转身离去。
<br/>　　显然，白管事提起了之前慕容胤四人说萧逸是小废物之事。
<br/>　　如今白家主承萧逸的情，自然对于慕容胤四人的言语感到很是厌恶。
<br/>　　“白家主，那是个误会阿。”慕容胤大急，却没有得到白家主的任何回应。
