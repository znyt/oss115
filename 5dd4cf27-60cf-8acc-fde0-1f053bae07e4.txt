　　没办法，谁让叶尘入校第一天，就刷爆了天海大学的论坛，成功占据了头条新闻，成为了天海大学的风云人物。
<br/>　　因而，众人对他的来历，全都好奇不已。
<br/>　　就连杨之凡和陶婉清，也纷纷扭头朝叶尘望了过来。
<br/>　　然而，
<br/>　　叶尘却依旧翘着二郎腿，丝毫没有要站起来的意思，淡淡开口道：
<br/>　　“我叫叶尘，天南省，云州人！”
<br/>　　众人等了半响，却发现叶尘已经又端起手中的酒杯，缓缓品了起来，不再言语，顿时尽皆愕然。
<br/>　　“这就完了？”
<br/>　　“也太敷衍了吧？”
<br/>　　“莫非是怕丢人，不敢说了？”
<br/>　　“肯定是这样！”
<br/>　　……
<br/>　　一时间，众人纷纷摇头。
<br/>　　尤其是在和杨之凡对比之下，越发觉得叶尘，根本无法和杨之凡相比。
<br/>　　杨之凡则两眼微微一眯，露出一抹疑惑之色，
<br/>　　在他看来，叶尘从头至尾都太淡定了，似乎根本没有把在场的任何人放在眼里。
<br/>　　按照他的经验，这样的人，要么狂妄无知，要么就是自身足够强大。
<br/>　　这时，
<br/>　　一旁的张连鹏站了起来，皮笑肉不笑道：
<br/>　　“叶尘同学，请你介绍的再详细一些，这样才能加深大家对你的了解嘛！”
<br/>　　“还有，介绍自己的时候，麻烦你站起来，这也是对大家最基本的尊重！”
<br/>　　然而，
<br/>　　叶尘却连眼皮都没有抬，淡淡道：
<br/>　　“你没有资格命令我该干什么，而且，我也不需要你们了解我！”
<br/>　　轰！
<br/>　　叶尘这话一出口，众人顿时全都愣住了，均没有想到，他居然敢当众顶撞学生会主席张连鹏。
<br/>　　张连鹏更是气的脸都绿了，差点忍不住就想要破开大骂。
<br/>　　杨之凡淡淡一笑，道：
<br/>　　“可能叶尘同学，不想别人知道他的身世，张学长何必和他一般见识？”
<br/>　　如果说之前，杨之凡还觉得叶尘深不可测，但是这番举动过后，已经彻底不把他放在眼中了。
<br/>　　但凡是有点脑子的人，都不会才刚刚踏入校门，就去得罪学生会主席。
<br/>　　听到杨之凡的一番话，张连鹏这才作罢。
<br/>　　很快，
<br/>　　联谊会继续进行，很多人纷纷开始向张连鹏敬酒，张连鹏得意之余，便将叶尘彻底抛之脑后。
<br/>　　至于其他人，看到叶尘已经把张连鹏得罪，自然也都躲得远远的，甚至连同桌几个人，也基本上不和叶尘四人交流，像是要把他们完全孤立。
<br/>　　“咱们再这么待下去，也没什么意思，要不我们自己去K歌吧？”
<br/>　　眼看着其他桌上，众人都已经开始谈笑风生，喝的一塌糊涂，李虎忽然开口提议道。
<br/>　　其他三人自然没有意见。
<br/>　　然而，
<br/>　　就在四人准备离场的时候，外面忽然传来一阵躁动。
<br/>　　继而，
<br/>　　一群人凶神恶煞的冲了进来。
<br/>　　“刚刚是谁特么扔的酒瓶？给老子站出来！”
<br/>　　“连我们鬼哥都敢偷袭，我看他是不想活了！”
<br/>　　这群人一冲入包厢之中，顿时便开始叫嚣起来。
<br/>　　而在众人中央为首的，是一名四十来岁的中年男子，一脸的煞气，头上还顶着几片瓶渣子，一道鲜血自头顶留下，显得有些狼狈。
<br/>　　原来，
<br/>　　是之前有人顺手把就酒瓶放在了窗沿上，方才玩闹的时候，不小心将酒瓶撞了下去，众人应该闹得太厉害，丝毫没有察觉，没想到竟然刚好砸中，从楼下经过的中年男子。
<br/>　　而且很不凑巧的是，他们的运气很不好，
<br/>　　这这中年男子也不是一个善茬，竟然是附近有名的混混头子，绰号鬼哥！
<br/>　　鬼哥的那帮小弟，看到自家大哥，竟然被人开了瓢，这如何还能忍？还以为是遇到了仇家，当即便带人冲了上来！
<br/>　　他们也没有想到，包厢里竟然是一群学生。
<br/>　　众人一时间还都没有反应过来，这到底是怎么回事，一个个吓得目瞪口呆，也没有人敢吭声。
<br/>　　“都特么听不懂人话是吧？刚刚那个酒瓶，到底特么是谁扔的？这个人最好自己乖乖站出来，否则你们这些人，今天一个也别想走！”
<br/>　　鬼哥脸上煞气十足，在众人身上扫视了一圈之后，一字一顿的开口道。
<br/>　　哗啦啦！
<br/>　　而随着鬼哥声音落下，其余众小弟，顿时纷纷掏出了各自的家伙。
<br/>　　“啊！”
<br/>　　在场的都是一些连架都没有打过的学生，何曾见过这种场面？顿时一个个吓得纷纷倒退，然后齐刷刷望向张连鹏和杨之凡。
<br/>　　毕竟现在，众人之中以两人为首。
<br/>　　杨之凡眉头微皱，不过却并没有要出头的打算。
<br/>　　张连鹏暗道一声倒霉，虽然很不想和这帮流氓混混打交道，可事到如今，他身为学生会主席，众人可都看着他呢，他自然不能退缩。
<br/>　　最终，
<br/>　　张连鹏一咬牙，只好硬着头皮走了上去，陪着笑脸道：
<br/>　　“这位就是大名鼎鼎的鬼哥吧！我想这应该是一个误会……”
<br/>　　砰！
<br/>　　还没等张连鹏说完，鬼哥旁边的一名小弟，直接从一旁抡起一个酒瓶，就狠狠砸在地上，酒瓶顿时瞬间粉碎，众学生个个脸色大变。
<br/>　　张连鹏也顿时吓得直哆嗦。
<br/>　　那大汉一个健步上前，一把便揪住了张连鹏的衣领，两眼一瞪，
<br/>　　“把我们老大打成这个样子，一声误会就完了？”
<br/>　　“大……大哥……那您……您说怎么解决？”
<br/>　　张连鹏被对方轻松制住，刚刚才鼓起的勇气，顿时一下子就怂了。
<br/>　　他之前就听说过，
<br/>　　眼前鬼哥这帮人，在天海大学附近这一带，向来嚣张跋扈，据说专门帮那些房地产公司，解决那些钉子户，手上握着不少人命，绝对个个都狠角色！
<br/>　　那大汉回头望向鬼哥，露出一抹询问之色。
<br/>　　鬼哥阴测测一笑，两眼在众人身上扫了一遍，缓缓道：
<br/>　　“看在你们是一帮穷学生的份上，想来应该也没胆子和老子作对！”
<br/>　　“只不过，老子这一下，也不能白挨！”
<br/>　　鬼哥说到这里，微微一顿，又在陶婉清、戴丽丽等人身上扫了一眼，脸上露出一抹淫笑道：
<br/>　　“男的可以滚了！女的留下，陪我们兄弟喝酒！”
<br/>　　鬼哥这话一出口，众人顿时纷纷脸色剧变，尤其是在场的女生，更是个个脸色惨白，甚至一些胆子小的，都险些吓晕过去……
