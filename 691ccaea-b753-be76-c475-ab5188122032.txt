　　第42章
<br/>　　攻击程大雷的兵卒不由停下脚步，仰望马上的程大雷心脏砰砰跳动。都说程大雷有五雷法，他们也略有耳闻，如果事实真是如此，那自己岂不是会被炸成飞灰。
<br/>　　砰！
<br/>　　墙头的黄三元和徐神机早已准备多时，此刻兜头浇下磨得细碎的辣椒面。红雾弥漫开，被吸如鼻中后又辣又疼，眼泪鼻涕顺着往下落。
<br/>　　“这……五雷法怎么是红色的？”高飞虎困惑道。
<br/>　　“这……这是火雷吧。”高飞豹。
<br/>　　“火雷……不该有火嘛？”高飞虎更加困惑。
<br/>　　“这……神仙的事，你懂什么。”高飞豹。
<br/>　　程大雷完全是将辣椒面当烟雾弹用，不得不说收到奇效，兵卒们连眼睛都睁不开，更遑论要发起攻击。
<br/>　　“杀！”
<br/>　　程大雷一声爆喝，与林少羽发起属于他们的冲锋。他们面对的都是步兵，在没有结成方阵的情况下，几乎无法对骑兵形成有效的阻挡。况且山坡上埋满陷阱，只要一离开安全区域，立刻就会陷入陷阱中。
<br/>　　士兵的战斗意志面临挑战，不停死亡的同伴正折磨着他们的意志。不由自主，大家都偷眼望着与秦蛮交战的陆哼，这是他们心底最后的旗帜。
<br/>　　陆哼的等级是优秀，当一个百夫长是完全够格的，三招棍法不过甩，砸，捅三个字，但仗着身大力沉威力也是不小。
<br/>　　可惜他今日遇见的是秦蛮。
<br/>　　秦蛮的等级是顶级，在升级之后，梅花枪已经悟到五蕊。一枪刺出，能在空中结五朵梅花，有虚有假，有真有实，若分不出枪里虚实，只有死一个下场。
<br/>　　况且说到力气大，秦蛮的力气也并不小呐。
<br/>　　三招两式，秦蛮便斜掉陆哼的怒气，毕竟技不如人，愤怒也无济于事。紧跟着秦蛮的大枪在陆哼耳边一晃，陆哼就感觉耳朵嗡嗡作响，秦蛮重重一拍，陆哼直接被砸落马下。
<br/>　　这一幕落在大多人眼里，距离太远也看不真切，有人惊呼一声，心中惊慌失措。
<br/>　　“陆将被杀死了！”
<br/>　　“陆将被杀了！”
<br/>　　“陆将死了！”
<br/>　　恐惧弥漫，伴随着战鼓的声音，士兵的战斗意志被摧毁。
<br/>　　人失其志，则无魂，兵失其志，则无战意……在战斗意志被彻底摧毁后，人群惶惶而逃，杀死他们并不比杀死一群猪困难多少，甚至更简单些。
<br/>　　这种事，已经是在古往今的战争中无数次被证实的事情。
<br/>　　陆哼就地一滚，在最短的时间上马，秦蛮提枪冲来，就要结果他的性命。
<br/>　　“且慢。”程大雷催马而来：“由我结果他！”
<br/>　　陆哼脸上肌肉跳动，可右臂的疼痛几乎令他连兵器都握不稳，他将铁棍交给左手，口中爆喝一声：“来呀！”
<br/>　　程大雷拍马靠近，两马像碰，大斧就抡了过去。
<br/>　　“我劈脑袋！”
<br/>　　这一招来势凶猛，可是在陆哼眼中也不过寻常，他举棍相挡，硬碰硬的撞击，可他忽略了自己用的是左手，力气减弱不少。
<br/>　　程大雷却是一招紧跟着另一招：“我掏耳朵！”
<br/>　　陆哼下意识向后仰身，两匹马马头相撞，陆哼直接被掀翻马背。
<br/>　　这是陆哼今天两次被打落马背，也是他这辈子仅有的两次，换做有气节的名将，当场自杀的也有。
<br/>　　真屈辱啊！
<br/>　　是的，今天他踏入青牛山时所施与给他人的屈辱，成倍的回到他身上。亲身体味后他会更明白，这滋味并不好受。
<br/>　　程大雷骑在马上，居高临下的望着他。
<br/>　　真特么令人讨厌的眼神呐！
<br/>　　陆哼站起，大吼道：“弟兄们，与我杀！”
<br/>　　“得了吧，你还哪有什么弟兄！”
<br/>　　陆哼一怔，茫然四顾，这才发现自己手下的士兵早已落荒而逃，地上留下的也不过是十几具尸体而已。
<br/>　　一个没有士兵的将军还是将军么？
<br/>　　陆哼没想到会落到这种境地，即便蛤蟆寨的挑衅有些出乎他预料，但平掉蛤蟆寨也不过是时间问题而已。但没想到，形势急转直下，直到如此。
<br/>　　沦落至此，或许只能一死而已。
<br/>　　他举棍向程大雷发起冲锋，被秦蛮一枪砸倒。他在站起来，再次冲锋，再次被砸倒，再冲锋，再砸到……
<br/>　　而马上程大雷注视着他，表情没有任何动容，但眼神却是越来越冰冷。
<br/>　　陆哼恍然明白，程大雷不是要杀死他，是要击垮他最引以为傲的东西。
<br/>　　凡自诩强大的就敲碎他给人看，凡自诩高高在上的就把他踩入泥淖。
<br/>　　而那些骄傲的，就要让他卑贱。
<br/>　　死，的确是洗刷屈辱的最好选择，可死也并不是一件容易的事情。
<br/>　　是屈辱的活着，还是尊严的死去，这是程大雷给陆哼的选择。
<br/>　　在数不清多少次冲锋后，陆哼选择了前者。
<br/>　　“回去给薛半川带个话，他想走那条路都可以，想从我蛤蟆岭下过，我不答应。”
<br/>　　陆哼昂起头，这是他保持尊严的最后方式：“你不怕黑石城的报复么？”
<br/>　　“让他们来。”
<br/>　　陆哼看了程大雷一眼，翻身上马，狼狈离去。
<br/>　　程大雷立在马上，目光扫过洒满鲜血的山谷，战斗留下的狼藉还在，如高飞虎一帮人也呆呆立在那里，每个人都是难以置信的表情。
<br/>　　他竟真的做到了。
<br/>　　不知何时，蛤蟆寨的墙头上出现两道人影，一男一女正是被擒在蛤蟆寨的李行哉和李婉儿。
<br/>　　注视着程大雷烈日下立马扛斧的身影，李行哉嘴角浮现一丝笑容。
<br/>　　“婉儿，还记得他写的那句诗嘛？”
<br/>　　“怎么能忘呢。”李婉儿望着李行哉的背影，缓缓道：“问苍茫大地，谁主沉浮。”
<br/>　　“是啊，怎能忘呢，不过九个字，若是传出去，不知会令帝国多少才子不再提笔。也不知全篇写出，是怎样的气魄。”李行哉又忍不住念了一遍：“问苍茫大地，谁主沉浮！”
<br/>　　黄三元来到程大雷身边道：“大当家，点清了，一共十车货，里面什么东西不知道，还有几批马。”
<br/>　　“把东西拉回山上，尸体找人埋了，人头……送到朱家寨给他们全尸。”
<br/>　　“是。”
