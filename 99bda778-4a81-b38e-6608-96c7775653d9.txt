　　第54章
<br/>　　“共一千贯，用马车拉来的。”
<br/>　　一枚铜钱大概4克重，一贯是一千文，大约4kg重量，一千贯就是4吨重量，还必须用马车拉来，一辆马车可能都不够。
<br/>　　这也是为何在如今用绢布当货币比用铜钱更方便的原因，在如今这个时代，还没有银票出现。
<br/>　　“够么？会不会有些少？”程大雷问。
<br/>　　“足够了。大当家，奴隶要比我们想象得便宜，成年奴隶不过一贯钱，老弱妇孺几吊钱就够了。之所以带这么多钱过来，一是柳芷赎身的钱要贵些，二是打点各方面，让交易顺利进行的钱。”
<br/>　　当今的物价，一匹马约二十贯，一头牛约十五贯，一头驴约七八贯。
<br/>　　四五贯就能买一个貌美如花的小丫鬟，成年壮劳力的价格不过一贯。
<br/>　　所以在这个时代，人没有驴贵。
<br/>　　“抓紧时间办事，今天下午就将柳家人赎回来，我们今夜就回去，路开得怎么样？”
<br/>　　现在山寨只有黄三元和徐灵儿二人，如果有人想打蛤蟆寨的主意，蛤蟆寨根本就是一座空城。
<br/>　　所以事情必须抓紧时间办，最好是今夜就能返回蛤蟆寨。
<br/>　　“落叶城晚上会关城门，第二天早上才会打开，不过我已打点后守城卫兵，他们会为我们开一炷香的时间，足够我们把人运出去。”
<br/>　　“那么就抓紧办事，军师、秦大哥和我去赎人，少羽你和孟子云去给柳芷赎身，我们在城外汇合。”
<br/>　　决定任务后，大家分头行动。令孟子云诧异的是，无论如何看，程大雷都是一个酒囊饭袋，可为何其他人都对他言听计从。
<br/>　　准备开始执行任务时，日头已经偏西，程大雷的计划是今夜就返回蛤蟆寨。时间紧，任务重，不过在有钱开路的情况下，这却也不算什么问题。
<br/>　　他以超出市价很多的价格赎回柳家的人，事情进行得很顺利，在夜幕初将时分已经将这些人接出城。
<br/>　　程大雷看到，这些人有小儿，有老人，有已经年老色衰的妇人，大概有八九十人之多。
<br/>　　当听到是蛤蟆寨的山寨救了他们后，人群掀起不小风波。
<br/>　　“老爷一辈子和山寨作对，我们怎么能去做山贼！”
<br/>　　“有辱祖上啊，有辱祖上啊！”
<br/>　　“我宁死不去做山贼！吾宁死，吾宁死！”
<br/>　　这一幕出乎程大雷的预料，而他也有些不耐烦。计划出了一些偏差，直到现在，孟子云和林少羽还没有回来。
<br/>　　该不会出什么意外了吧！
<br/>　　孟子云的性命他不在乎，柳芷他也没有那么在乎，可林少羽却是山寨的人，他不得不在乎。
<br/>　　纷纷攘攘的声音在周围响起，程大雷烦不胜烦。
<br/>　　“都住了吧！”程大雷喝了一声，将腰中刀丢在地上：“不想做山贼的可以去死，死的法子也有很多，不想用刀的，可以找石头撞死，找大树吊死，这不是你们显示忠君报国，高人一等的时候，想死的，自便。”
<br/>　　没人说话。
<br/>　　不管是那些叫嚣着士可杀不可辱的老夫子，还是自诩一腔热血的少年，都没人说话。
<br/>　　如此，换来的只能是程大雷的鄙夷。
<br/>　　他将目光望向城门方向，心中暗暗有些焦急，与预估时间相比，林少羽迟到时间太久了。
<br/>　　这时，城门方向有一匹快马奔来，马上的人是孟子云。
<br/>　　“什么情况，林少羽和柳芷呢？”程大雷立刻问。
<br/>　　“小姐……今天城内黄家有家宴，小姐被请了过去，我们去的时候，小姐已经被带走，林少羽让我回来报告消息，他自己想去看看。”
<br/>　　柳芷在回燕楼挂牌，想要见她规矩很多，可真的碰上有权有势的人，什么规矩都得作废。黄家要她去，她也不得不去。
<br/>　　这黄家似乎就是要娶苏樱的黄员外。
<br/>　　“然后你就自己回来了？”程大雷看着孟子云，嘴里吐出两个字：“蠢货！”
<br/>　　一听这话，孟子云一张脸腾地红了，怒道：“小子，我忍你很久了，你当自己是什么人，有什么资格教训我，再敢对我有丝毫不敬，我认得你，我的刀可不认得你。”
<br/>　　这孩子有些太不懂事了，该是教教他什么叫规矩。
<br/>　　“秦蛮，动手。”
<br/>　　“是！”
<br/>　　秦蛮拍马向前，孟子云早就对白日发生的事情耿耿于怀，想要找个机会给这几个山贼一些颜色看看，此刻刀已经亮了出来。
<br/>　　可秦蛮连枪都没有动，在两马相错的刹那，抓住孟子云的衣领，直接丢了出去。
<br/>　　秦蛮两臂至少有五百斤力气，这一下发力，拎着孟子云在空中头四肢悬空，身体在空中打了个转，直接丢出七八米远，硬生生的砸在地上。
<br/>　　一片安静，有人慌张将孟子云扶起，孟子云虽然没有受伤，却是羞得无地自容，他向来骄傲，遇到这样的事，却还是平生第一次。
<br/>　　他立在地上，双目如火般盯着程大雷，这样的侮辱让他恨不得与程大雷同归于尽。
<br/>　　“柳家的人都是这样没有规矩么？”程大雷骑在马上问：“再有下次，杀无赦。”
<br/>　　程大雷第一次展现自己的杀意，他先以无人击千，后又挡住陆哼一百精兵，居移体，养移气，他早已不是孟子云以为的那个酒囊饭袋，而的的确确是位啸傲山林的一方贼首。
<br/>　　“大当家，现在少羽还在城内，我们怎么办？”
<br/>　　“这要是进了城，再出来怕就难了。”徐神机道。
<br/>　　这个时候有个柳家的老人忽然道：“我知道一条通向城外的路，哪里城墙缺了个豁子，且没有卫兵把守。”
<br/>　　程大雷问清楚那小路的位置，心中还在思索。这帮人长久留在城外，迟则生变，必须尽快将他们送走。而林少羽还耽搁在城中，却又不得不管。
<br/>　　他心中已经隐隐有了主意，只是还没下定决心。
<br/>　　徐神机似乎猜到程大雷在想什么，冲他道：“大当家，从山寨来的时候，我把你的斧头也带来了，就在马车上。”
<br/>　　说着，徐神机把那柄大斧给程大雷取来，程大雷提着自己的宣花斧，心中多了几分自信。
<br/>　　“秦蛮，孟子云，随我进城，抢人！”
<br/>　　
<br/>(本章完)
