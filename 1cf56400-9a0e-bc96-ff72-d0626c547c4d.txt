　　第340章
<br/>　　云中龙等人今天的心情如波浪线一样，起起伏伏。初见程大雷时因为恐惧而紧张，见到程大雷如此随和，心情都有些放松。再到后来，大家说什么程大雷听什么，心底难免有些轻视。
<br/>　　丛林法则下，宽容换来的必然不是尊重，而是不屑。
<br/>　　终于到现在，当程大雷露出了锋芒，众人的感受是不寒而栗。
<br/>　　什么送来读学堂，根本是将骨肉送到琴川关做人质，这样狠的手段，就连莫鸣米都未做过。
<br/>　　如此说来，这是一个要比莫鸣米更加狠毒的存在。
<br/>　　“有困难？”程大雷看着众人闪烁的目光：“当然啦，世道不太平，路上也不安全，觉得有困难的我可以派人去接。”
<br/>　　“不必，不必。”
<br/>　　“其实我早就想将我儿子送到学堂了，只是一直没有机会。”
<br/>　　“跟在程当家身边，他如果能学个一招二式，也就够他一辈子受用了。”
<br/>　　“这是多少人求都求不来的机会呐。”
<br/>　　程大雷很是满意眼前这一幕，点点头道：“那么这件事就愉快的决定了。”
<br/>　　“愉快的决定了……”
<br/>　　“愉快，很愉快。”
<br/>　　大家说着这话，脸上挤出的都是苦笑。把儿女送到程大雷手中，那么大家就必然被程大雷绑在一起，如果莫鸣米能想到这一招，他恐怕也不会败得这么惨。
<br/>　　“然后是聊聊大家以后吃饭的问题？”程大雷道：“以后路过的商队，就不要向他们收保护费了。”
<br/>　　“没有保护费，我们吃什么？”云中龙问。
<br/>　　程大雷很满意他的接话，这都是事先安排好的。程大雷故意板起脸：“现在琴川关还有商队么？”
<br/>　　“这个……已经很久没人来了。”
<br/>　　“所以嘛……”程大雷拍拍手：“司马星进来。”
<br/>　　留着两撇胡须，看上去很像奸商的司马星走了进来。
<br/>　　“给各位介绍一下，这位是司马星，是我刚任命的商业总管，以后大家吃饭的问题，就由他负责。”程大雷道：“司马星，你给大家聊聊以后的规矩吧。”
<br/>　　司马星轻轻嗓子，他从未被如此重视，今天还特意打扮了一下。面对众人的目光，他缓缓开口：
<br/>　　“目前的琴川很荒凉，地广而人稀，因为以前一些事情，商队也不敢往这里来。没有人哪里来的财，所以首先要做的是把商队引进来，大家应该想办法把商队引进来，不仅不向他们收取保护费，还要给他们一定优惠……大当家教会我一个新词：招商引资。”
<br/>　　程大雷点点头，很满意今天司马星的表现，这司马星看上去比吴用好使啊。
<br/>　　“所有人都要去宣传，告诉他们琴川已经和以前不一样了，在这里不仅不会被抢劫，谁如果被抢劫了，只要在琴川地界发生，我们还会补偿损失。等日后琴川繁荣起来，我们再开始收税。”
<br/>　　“还有一件事……”司马星拉了一个长音：“琴川地广人稀，很多土地都很肥沃，却很肥沃。大家要想办法把人拉过来，有了人才能收取更多的税收。目前帝国流民是多的，只要给他们一块土地，不受欺压他们都是愿意来的。”
<br/>　　程大雷补充道：“这件事可以稍微大胆一些，能骗就骗，能抢就抢，只要不太过分就可以。毕竟，咱们周围还有很多城池，虽然目前大家还是井水不犯河水，可谁知道以后怎么样。”
<br/>　　这些话云中龙等人已听不太懂，觉得还是以前爱抢谁抢谁时痛快。现在这些，操作起来太复杂了。
<br/>　　程大雷叹口气，整体素质太差，完全带不起来呀。
<br/>　　“从今天开始，每家轮流到琴川关拉练，想想你们以前的样子，也配叫一个兵。”程大雷板起脸：“该流些汗，吃点苦，只有先苦才能后甜。”
<br/>　　众人虽然不太明白程大雷的话，但却是不敢不听。此刻见程大雷已经有些发怒，顿时齐声道：“一切听大当家吩咐！”
<br/>　　程大雷差人送酒过来，率先割破手指，滴入酒中。十八家头领轮流做了，每人手中都端着血酒。
<br/>　　“为了美好的明天，干。”
<br/>　　众人举起酒碗，同时一饮而尽，至于程大雷所说的美好明天，大家心里怕一个字都不信。嘿，能活下来就算不错了。
<br/>　　一切都按程大雷设计的发展，生活职业的加入，让百废待兴的琴川关迈入正轨。
<br/>　　十八家首领开始搬家，他们的子女也都送到琴川关。有些已经二十余岁的汉子，有些还是七八岁的黄口小儿。
<br/>　　程大雷将他们安排到学堂，由吴用带着先生给他们上课。在学堂入学的不仅有孩子，还有许多五大三粗三十几岁的汉子。
<br/>　　不过他们的文化程度，和孩子也差不了多少。
<br/>　　及到此时此刻，程大雷才算彻底控制琴川关，内忧外患都已理清，他成了八百里琴川实际控制者。
<br/>　　而在程大雷忙碌的时候，生四级本的工作也在进行，因为琴川关没有建筑精通的技能，所以工程进度要慢一些。但忙碌这么长时间，工程也终于到了完工的日子。
