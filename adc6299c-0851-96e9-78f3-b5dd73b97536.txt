　　第230章
<br/>　　林少羽昂起头，目光向看台上看。
<br/>　　他看向明帝时，明帝微闭双目；他看向尉迟离时，尉迟离将头扭到一边；他看向崔相时，崔相以手扶额，陷入假寐。他看向台上的文官武将，皇亲贵胄，他们视若无睹，仿佛没有看到林少羽这个人。
<br/>　　反而是金问道站起身，道：“林少侠，许久未会，为何落魄如此？”
<br/>　　林少羽没有理他。
<br/>　　刘阿吉站出来将手中拂尘一甩：“禁卫军，将扰乱武举的人押下去。”
<br/>　　有禁卫军围上来，他们仍忌惮林少羽刚才展现的武艺。但林少羽并没有反抗，他早已放下手中枪，只是口中高声喊着。
<br/>　　“家父林问天，赤胆忠心，天地可鉴，为何却身首异处，遗臭万年，不孝子林少羽要替家父喊冤，还他一个公……”
<br/>　　他的嘴巴已经被人堵住了，后面都话他就说不出口了。一心想要公道，可有时候连公道这两个字你都说不出口。看着他狼狈的样子，程大雷既没有出手的机会，也不想出手。
<br/>　　看台上的诸人似乎又恢复清醒，林少羽的出现的确有些尴尬，可他们早学如何看不见尴尬的事情。
<br/>　　今日武科举算是被戎族毁了，估计明帝是没脸再封屈九江或者柳轻名，假若林少羽不是林少羽，倒还可以把他提为状元，可惜事不遂人愿。
<br/>　　那么今日的事情就剩下两项，一是封赏杨龙停，二是封赏程大雷。
<br/>　　还有许多仪式要走，鸣鼓，响炮，禁卫军要展现一下仪仗，趁这个时间，大家还要商议一下武状元的事情究竟如何办，是干脆不封了，还是随便提个人上来，遮住丢掉的面子。
<br/>　　“陛下，明玉公主特意命戏子排了一出戏，代替原先的剑戟舞，陛下要不要看一下？”
<br/>　　“婉儿的主意？”明帝略微来了一些精神：“那就看一下吧。”
<br/>　　其实所有人都对这样繁琐的仪式烦不胜烦，可这样的场合你不表现一下虔诚也是不行，在这个时候，有出戏能打发一下时间也是好的。
<br/>　　演武场内早有高台，事先一布置成戏台，有幕布分出前后台。
<br/>　　明帝笑道：“看来婉儿早就有准备。”
<br/>　　“明玉公主是很有心的。”刘阿吉道。
<br/>　　当明帝露出感兴趣的模样时，在场又有那个敢没有兴趣。一时间，所有人的注意力都集中在戏台上。
<br/>　　在所有人的目光中，楚青衣一身白袍走上舞台中央，他脸扮得很白，眼中没有任何表情。一时间竟没人能分辨出他是男是女，或者非男非女。
<br/>　　据说菩萨也非男非女。
<br/>　　如此大家的心神情不自禁被他吸引，目不转睛的盯着他一举一动。
<br/>　　“极北之地的魔王已经苏醒
<br/>　　天寒地冻又碰上万里雪封
<br/>　　一场刀兵之祸即将到来
<br/>　　世人还身在梦中浑然不知”
<br/>　　他的唱腔和长安流行的曲调完全不同，与其说是唱腔倒不若是咏叹，伴随着洞箫，声音空灵，若是天籁。
<br/>　　舞台上有黑衣的魔族挣扎着站起，他们青面獠牙，狰狞可怖，一座城池被攻破，身着布衣的平民被屠杀，身着锦衣的贵人在逃亡。
<br/>　　楚青衣倒在地上，挣扎，扭曲身体，身上的白袍被他自己扯烂，脸上作出痛苦的表情。
<br/>　　谁都明白他在表演什么，他在表演被强暴，那如蛇般扭动的身体，绝望欲死的表情，纵然只有一个人，却仿佛让人看到一头粗鲁肮脏的恶狗在玷污月亮。
<br/>　　观众的心都被揪了起来，是一个女子被强暴么，不，是整个帝国在被强暴。
<br/>　　一个身着盔甲的将军在逃亡，他路过被强暴的楚青衣，被屠杀的平民，目不转睛，仓皇逃跑。
<br/>　　楚青衣缓缓站起，他从不喜不悲的菩萨化作衣衫凌乱的弱女子，依旧是一张惨白的脸，面无表情。
<br/>　　“耕耘的田地被铁蹄踏破
<br/>　　牺身的屋内燃起大火
<br/>　　幼子的尸体悬在房梁
<br/>　　而我也成了不洁之人
<br/>　　该保护我们的人身在何处
<br/>　　却原来他逃跑时连风也赶不上
<br/>　　为何有这样的灾祸
<br/>　　是天要降苦难在人间
<br/>　　还是神并不怜悯世人”
<br/>　　鼓声忽疾，点点敲在人心脏上，令观众的情绪随着紧绷。
<br/>　　在魔族追杀的道路上，突然出来一队青衫人。
<br/>　　楚青衣持了一柄剑，气势瞬间一变，刚才若说他是个妩媚较弱的女子，此刻他就是世间最铁骨铮铮的男儿。
<br/>　　冷目绽寒光，铁剑灿杀机，他虽只有一人，却宛若在与千军万马搏杀。
<br/>　　身后青衫与黑衣搏斗，青衫的人数虽少，如同在黑暗的大海中被冲击，可一波波攻击，却被守住了。
<br/>　　鼓声，洞箫声，琵琶声……越发急促，而舞台上的战斗也愈发激烈。
<br/>　　舞台明显被分为两半，另一面一位身着盔甲的老将陷入黑衣人的包围中。
<br/>　　楚青衣舞剑，边舞边歌：
<br/>　　“岂曰无衣？与子同袍
<br/>　　岂曰无衣？与子同泽
<br/>　　岂曰无衣？与子同裳”
<br/>　　一歌三叹，曲调哀婉，伴随着铮铮的鼓声，将所有人的注意力凝在一起。大家的情绪不自觉随舞台上的表演而起伏，关心青衣人能不能守得住，关心那老将能不能等来支援。
<br/>　　舞台的变幻突然加快，厮杀，厮杀，厮杀，同时舞台的角落中，一个小丑模样的人冷哼着，拦住将要送达的粮草。
<br/>　　那一边逃跑的将军窝在城头，鬼鬼祟祟向外探出脑袋。
<br/>　　如果这下大家都不明白是怎么回事，那么帝国的文武官员就都成了傻瓜。崔相一张脸已经变得极其难堪，而杨龙停反而是面无表情。
<br/>　　大家都看向明帝，觉得戏不能再演下去了，可他在场，大家都不能替他拿主意。
<br/>　　明帝微微睁开双目，手敲着龙椅，缓缓道：“有点意思么。”
<br/>　　众人都收回视线，既然明帝说有点意思，那大家都得觉得有意思。可是这样一出戏，何止是有点意思，里面意思大了。
<br/>　　金问道腾地一下站起来，目光环视四周，心脏在砰砰打鼓。
<br/>　　杨龙停此刻和他是同样感觉，以目光扫着演武场。
<br/>　　那个人一定在这里，他没有死，他还活着，而且他就在自己身边。
<br/>　　问题是，他究竟在哪里，或者说……
<br/>　　他究竟是谁？
<br/>
