<br/>    阴月天狼本是率军前去攻打一位大妖王，却是被最新接到的消息给震惊了，“什么，燕山居然占了上风？”
<br/>    
<br/>        麾下有妖王询问道：“大人，我们还去攻打吗？”
<br/>    
<br/>        阴月天狼抬头看天，然后道：“打？凭什么打？这妖族的天下，已经变了天了。这陈明，简直阴险狡诈之极，世人都以为他只是大妖王的实力，而他真正的实力，却是大君！速速撤回妖城，恐怕这一次，必须要大君亲自出手了。”
<br/>    
<br/>        陈明这一段时间，正在缓慢行军，妖族大君至少拥有二十万妖军，甚至是三十万，他想要等其他五路大军铲平这些大妖王之后汇兵一处之后再去攻打妖族大君。
<br/>    
<br/>        这一战，陈明要的很简单，十万大山的地盘，他要，他播下了赤潮的种子，那就要给这些种子一片生长的地盘，那就是十万大山。
<br/>    
<br/>        妖族大君的命，他也要，一位活着的敌对大君，对于谁来说都是一个莫大的威胁，评级灾难的存在，可不是弄着玩的。
<br/>    
<br/>        陈明此刻，正在和紫霞仙子讨价还价，“我说，我帮你突破到了大君，我不就问你要几部功法嘛，你至于吗？”
<br/>    
<br/>        紫霞仙子正色道：“功法乃是一个宗门最为紧要的东西，我绝不能给你。”
<br/>    
<br/>        正在这时，忽然有血衣剑卫来报，“启禀山主，妖族大君调回了阴月天狼，已经在召集兵马，似乎是准备袭击我们！”
<br/>    
<br/>        陈明道：“这个时候才反应过来吗？”
<br/>    
<br/>        紫霞仙子道：“那妖族大君有着多少兵力？”
<br/>    
<br/>        陈明道：“三十万妖军，他应该是有的，你也知道，妖族几乎都在十万大山，又只有一位大君，他的兵力，绝对不是小数目，而且，他投靠了鬼仙，所以将自己的兵马扩充到了三十万，就是为了一统十万大山。”
<br/>    
<br/>        紫霞仙子道：“我们现在这里不过两万五千大军，若是那妖族大君挥兵来攻，我们要怎么办？而且，对方可是有着王级军团的，燕山，可没有王级军团。我们现在唯一的优势，就是我们拥有着两位大君。”
<br/>    
<br/>        陈明道：“所以说，我们现在只能等其他五路大军铲除了其他所有大妖王之后，才有机会和妖族大君决一死战。”
<br/>    
<br/>        紫霞仙子道：“我是真的不明白，你是怎么想的，每个大妖王，麾下都是至少两三万的妖军，你居然兵分五路，一路只有一万人，唯有卓青瑶那一路，有着三万人，可是卓青瑶面对的敌军，只怕不下十万。我现在很怀疑，你是怎么坐上燕山之主的。”
<br/>    
<br/>        陈明随意摆了摆手道：“要是你来指挥，你会怎么打？”
<br/>    
<br/>        紫霞仙子道：“自然是集兵于一处，一个大妖王，一个大妖王的覆灭！灭掉所有大妖王之后，然后靠着收服的妖军，应该有二十万左右的兵力，这样一来，便可以和妖族大君分个胜负了！”
<br/>    
<br/>        雪自悟也是道：“宗师，若不是在九幽寒泉之中你带领我们打赢了，我现在很怀疑你的决策。”
<br/>    
<br/>        陈明看了一眼雪自悟，骂道：“和苏青阳下你们的飞行棋去！”
<br/>    
<br/>        陈明继续道：“若是按照你们的方法，不是不行，而且很稳妥，不过有两点，让我很不满意。”
<br/>    
<br/>        苏青阳问道：“哪两点？”
<br/>    
<br/>        陈明道：“第一点，这样很慢，按照你们这个思路一路打过去，怕不是要打一年。第二点，若是不分兵，给妖族大君看到燕山的实力，他不一定会和我们一战，他归属了鬼仙，完全可以投靠十万大山附近归属了鬼仙的其他大君，找到机会，再打回来。”
<br/>    
<br/>        紫霞仙子瞬间明白了过来，“你的意思是，你就是要妖族大君前来进攻，只有这样，才有机会斩杀他！”
<br/>    
<br/>        陈明点头道：“在这个大陆上，大君就是最强大的力量，他们拥有着毁灭一切的力量，这样的存在，出现一位新君，是值得通报整个大陆的大事，死去一位大君，同样是震惊整个大陆的大事，想要杀一位大君，可没有那么容易。”
<br/>    
<br/>        紫霞仙子道：“可是现在，我们怎么守得住？”
<br/>    
<br/>        陈明笑了笑道：“妖族大君的实力，应该和你差不多，都不怎么样，不然他要是有我这个实力，早就一统十万大山了，到时候，你去缠住妖族大君，其他的，就交给我了，群战什么的，我最擅长了，要知道，在这十万大山，我可是有着一个称号，大阵之下，众生平等。”
<br/>    
<br/>        紫霞仙子对于陈明嘲笑自己的实力并没有什么好脸色，“哼，不就是实力强一点点嘛，至于这么嚣张吗？你那五路大军打不打得过都是个问题，还说这些。”
<br/>    
<br/>        陈明继续道：“当年银翼带着燕山镖局的一万镖师，都能把一位大妖王灭了，燕山的血衣剑卫，比燕山镖师精锐三倍，灭了几个大妖王还不是信手拈来。”
<br/>    
<br/>        紫霞仙子道：“我就看看你们燕山者血衣剑卫，到底有多厉害！两万五打三十万，怕不是不知道怎么死，反正我是大君的修为，想走就走。”
<br/>    
<br/>        苏青阳和雪自悟也是道：“也对，以我们的实力，跑还是没问题的。”
<br/>    
<br/>        陈明喝道：“你们两个和她能一样吗？人家是已经完成了杀人证道，所以能跑，你们这一次跑了，道心蒙尘，我看你们这辈子都别想突破到大君了，修行本就是与天争命，天且不惧，还惧人乎？”
<br/>    
<br/>        苏青阳和雪自悟现在也终于是醒转过来，苏青阳拱手道：“多谢宗师教诲，我等，险些误了大道！”
<br/>    
<br/>        雪自悟更是眼中闪现出蓬勃的战意，“不错，天且不惧，还惧人乎？”
<br/>    
<br/>        正道仙门，还是这样好忽悠。
<br/>    
<br/>        陈明扫了一眼自己的面板，距离大世开启还有一个月，现在整个大陆可谓是风云变幻，就连占了天下半壁江山的魔宗都入场了，仙魔大战估计还有着一年多的时间，毕竟我的剧本上写的是大世开启之后一年才是斩魔大会，那么这个大世开启，到底是指的什么呢？
<br/>    
<br/>        估摸着是那四个古仙门老君商量出来的什么事吧。
<br/>    
<br/>        留给自己的时间，可不多了，自己也只能冒险想办法越快越好了。
