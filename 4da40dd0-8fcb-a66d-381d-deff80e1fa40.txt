　　叶尘没有吭声，一脸淡漠，
<br/>　　现在的他，的确没有绝对把握杀死绝无敌，可如果仅仅只是自保的话，还是绰绰有余的。
<br/>　　“来接我第一招！”
<br/>　　另外一边，绝无敌已经悍然出手了。
<br/>　　只见他站在原地没有挪动脚步，只是随手一拳砸出。
<br/>　　轰隆！
<br/>　　一道携带着惊天霸气的硕大拳头，犹如一座小山般，朝叶尘狠狠砸落了过来！
<br/>　　“嗯？”
<br/>　　一旁的龙辰看到这一幕，顿时眉头微微一皱。
<br/>　　在他看来，绝无敌这一拳，应该只用了三四成的功力，可是对付一个只有元婴境的小子，还是有些过分了，对方就算不死，恐怕也得重伤！
<br/>　　他并不关心叶尘的死活，可是如果一旦有人死了，极有可能会影响到迷罗星海的开启，所以在通往迷罗星海的通道没有开启之前，他不允许任何人出事。
<br/>　　就在龙辰心中暗暗打定主意，一旦叶尘无法抵挡，他就第一时间出手想帮的时候。
<br/>　　叶尘同样也出手了，而且也是同样一拳砸出，
<br/>　　“虚空神拳！”
<br/>　　相比于绝无敌的霸道一拳，叶尘这一记虚空神拳，看上去轻飘飘的，似乎毫无力道。
<br/>　　而龙辰却双眸之中，却闪过一抹惊色，
<br/>　　“虚胜实，不足胜有余！好精妙的拳法！”
<br/>　　龙辰毕竟是天骄榜首，眼光自然不会太差，一眼便看出了，叶尘这招拳法中的精妙之处，心中又惊又奇。
<br/>　　周围其他人，就没有这样的眼光了，不由得纷纷摇头，
<br/>　　“别说三招，就连一招，这小子也撑不住啊！”
<br/>　　“不自量力！”
<br/>　　“绝无敌不愧是天榜前十，随手一击，竟然恐怖如斯！”
<br/>　　……
<br/>　　轰隆隆！
<br/>　　在众人的议论声中，相差悬殊的两个拳头，在空中重重相撞在一处。
<br/>　　然而，
<br/>　　下一刻，
<br/>　　绝无敌那看似霸道无比的一拳，竟然消散于无形！
<br/>　　“什么！！”
<br/>　　就连绝无敌，也不由得微微一愣，继而又猛然感觉到，一股强大的拳劲，似乎穿梭空间，朝他的面门狠狠冲击而来！
<br/>　　蹬蹬蹬！！
<br/>　　由于事发突然，有些反应不及，绝无敌几乎是下意识的倒退了数步，然后方才运起护体真元，化解了那道拳劲。
<br/>　　哗！
<br/>　　周围众人却并不知道其中诀窍，他们只看到，明明占据绝对优势的绝无敌，竟然退了！
<br/>　　而那个只有元婴境巅峰的小子，从头至尾，都微丝不动的站在那里！
<br/>　　一时间，全场顿时一片哗然！
<br/>　　任谁也没有想到，两人的第一招比试，竟然以绝无敌落败而告终。
<br/>　　就连那云破天、燕狂生等几大天骄，也全都满脸惊愕之色。
<br/>　　甚至，之前看出叶尘拳法精妙的龙辰，也没有想到，叶尘看似简简单单的一拳，竟然还蕴藏了后招，一举将绝无敌逼退了！
<br/>　　“原来是将两道拳劲，融入了一拳之中，而且还融入了空间法则，打了绝无敌一个措手不及，有意思！难怪他口气这么大，看来是我小瞧他了！”
<br/>　　龙辰深深地望了叶尘一眼，眼眸深处，透着一抹浓浓的好奇之色。
<br/>　　“绝无敌，你也就这点能耐吗？”
<br/>　　叶尘虽然有投机取巧的嫌疑，可终究还是赢了，脸上挂着一抹戏谑之色，淡淡嘲讽道。
<br/>　　“混蛋！你找死！！”
<br/>　　绝无敌被叶尘当着这么多人的面出言奚落，顿时气得肺都要炸开了。
<br/>　　他刚刚完全是被叶尘给算计了，根本没有想到，对方一拳之中，竟然包含两道拳劲，一时大意，这才着了对方的道。
<br/>　　虽然他根本没有受伤，但在众人眼中，却是他落下了下乘，这让他有一种，一口老血憋在胸口，却又吐不出的郁闷。
<br/>　　轰！
<br/>　　一股惊人的冲天气势，从暴怒的绝无敌身上，陡然升腾起来，居然汇聚成了一道高达上千米的外道神像，几乎都快要凝聚成实质，而且煞气冲天！
<br/>　　蹬蹬蹬！
<br/>　　周围一些修为弱小者，看到这一幕，顿时纷纷倒退到高台边缘的位置，生怕被那股强大的气势所波及到。
<br/>　　很显然，在被叶尘奚落之后，绝无敌是真的怒了，恨不得要立即宰了他，以报刚才的羞辱。
<br/>　　就在这时，一旁的龙辰，忽然开口了，
<br/>　　“绝师弟，莫要忘了刚刚你答应过我什么！”
<br/>　　绝无敌听到龙辰这话，顿时微微一滞，略微恢复了几分理智，冷冷一笑，道：
<br/>　　“龙老大放心！我不会这么轻易就杀死他，我只是要让他明白，他和我之间的差距，究竟有多大！！”
<br/>　　说完这话，绝无敌缓缓抬起手臂，朝叶尘一拍！
<br/>　　轰隆！
<br/>　　随着绝无敌一掌拍落，那高达千米的外道神像，也抬起了巨大的手掌，如同一座大山般，朝叶尘压迫了下来！
<br/>　　绝无敌似乎是故意放慢了出手的速度，显然是想要逼迫叶尘倒退，让众人也看到他狼狈逃窜的模样。
<br/>　　可惜，他失望了，叶尘不但没有退，反而一步跨出，主动迎了上去，
<br/>　　“逆生百转！！”
<br/>　　直接将逆生百转神通，运转到了第六重境界，对于如今的叶尘来说，简直轻而易举。
<br/>　　在功力暴涨的同时，叶尘再次一拳砸了出去，
<br/>　　“八极镇魔神拳！！”
<br/>　　轰！轰！轰！轰！
<br/>　　八道硕大的拳影，自四面八方，朝绝无敌的外道神像轰击而去！
<br/>　　轰隆隆！
<br/>　　绝无敌那道高达千米的外道神像，在叶尘这一记八极镇魔神拳的轰击之下，顿时一阵颤动，几乎险些破碎开来！
<br/>　　蹬蹬蹬！
<br/>　　绝无敌遭到一股反噬之力，顿时再次一连倒退了数步，原本那霸气冲天的外道神像，竟然也虚淡了几分！
<br/>　　“什么！！”
<br/>　　看到这一幕，在场的所有人，顿时全部都惊呆了，就连龙辰也不例外。
<br/>　　如果说，两人第一次交手，绝无敌是因为大意轻敌，所以才会着了叶尘的道，可是这一次呢，叶尘却是正面对抗，再次将绝无敌逼退！！
<br/>　　而且，龙辰可以清晰的感受到，这个明明只有元婴境巅峰的年轻人，气息突然爆炸式的增长，修为仿佛一下子提升了数十倍！
<br/>　　这所有的一切，都太不可思议了！
